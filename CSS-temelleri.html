CSS temelleri

CSS (Cascading Style Sheets), web içeriğini şekillendiren koddur. CSS temelleri, başlamak için ihtiyacınız olan şeyleri açıklar. Şuna benzer soruları yanıtlayacağız: Metni nasıl kırmızı yapabilirim? İçeriği (web sayfası) düzeninde belirli bir konumda nasıl görüntüleyebilirim? Web sayfamı, arka plan resimleri ve renkleri ile nasıl süsleyebilirim?

CSS Nedir

HTML gibi CSS de bir programlama dili değildir. Bir işaretleme dili (markup language) de değildir. CSS bir stil sayfası dilidir. CSS, HTML öğelerini seçime bağlı olarak biçimlendirmek için kullandığınız şeydir.

Bir CSS kural setinin anatomisi

Nasıl çalıştığını anlamak için kırmızı paragraf metninin CSS kodunu inceleyelim:

Bütün yapıya kural seti denir. (Kural seti terimi genellikle sadece kural olarak adlandırılır.) Tek tek parçaların adlarına dikkat edin:

  • Selector: Bu, kural setinin başlangıcındaki HTML öğesi adıdır. Stillendirilecek eleman(lar)ı tanımlar (bu örnekte "p" elemanları). Farklı bir öğeye stil vermek için selector'ü değiştirin.
  • Declaration: Bu, color: red; gibi tek bir kuraldır. Öğenin özelliklerinden hangisini stillendirmek istediğinizi belirtir.
  • Property: Bu, bir HTML öğesini stilize edebileceğiniz yöntemdir. (Bu örnekte renk, "p" öğelerinin bir özelliğidir.) CSS'de, kuralda hangi özellikleri etkilemek istediğinizi seçersiniz.
  • Property value: Property'nin sağında - iki nokta üst üste işaretinden sonra - property value vardır. Bu belirli bir property için birçok olası görünümden birini seçer. (Örneğin, kırmızıya ek olarak birçok renk değeri vardır.)
